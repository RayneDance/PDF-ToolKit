# Install on Windows

The recommended path for most users is the Windows installer from GitHub Releases.

## Steps

1. Download `pdf-toolkit-setup-windows-x64.exe` from the latest release.
2. Run the installer.
3. Keep the default per-user install path unless you have a reason to change it.
4. Choose whether you want the optional desktop shortcut.
5. Launch `PDF Toolkit` from Start Menu after setup completes.
6. Start with the `Start Here` screen and pick a workflow template such as `Merge Invoice Packet` or `Watch Incoming Folder`.

## What the installer includes

- Core desktop app only
- Start Menu shortcut and normal Windows uninstall entry
- Optional desktop shortcut
- No `.pdf` file association changes
- No OCR dependencies bundled
- No background updater or account setup

## Unsigned installer note

The first public installer release is unsigned. Windows SmartScreen may show a warning before launch.

## What works right away

- Combine, split, preview, and inspect PDFs
- Export text, images, attachments, metadata, and tables
- Use bundled workflow templates from the `Start Here` surface
- Run folder workflows locally with repeatable outputs

## What is optional

- OCR support
- OpenAI-powered LLM analysis

## Portable fallback

If you prefer not to install the app, download `pdf-toolkit-windows-x64.zip`, extract it to a normal folder, and run `pdf-toolkit-gui.exe`.

## What works immediately

- Merge, split, rotate, crop, preview, render, extract text, metadata, forms, attachments, tables, and automation workflows.

## Optional OCR

OCR is not bundled in the default public release yet. See [enabling-ocr.md](./enabling-ocr.md) if you want to add OCR support later.
